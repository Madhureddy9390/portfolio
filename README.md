# Personal Portfolio Website

A modern, responsive portfolio website built with **React.js** to showcase my professional experience, technical skills, featured work, and contact information. The application focuses on clean UI, accessibility, reusable components, and a seamless user experience across desktop, tablet, and mobile devices.

## Live Demo

🔗 https://portfolio-pink-psi-25.vercel.app/

## Features

- Responsive design for desktop, tablet, and mobile
- Light and Dark theme with persistent user preference
- Professional Hero, About, Skills, Experience, Projects, and Contact sections
- Featured work showcasing professional and personal projects
- Reusable component-based architecture
- Smooth scrolling navigation
- Active section highlighting
- EmailJS integration for contact form submissions
- Accessible UI with keyboard-friendly navigation
- Modern animations and hover interactions
- Clean and maintainable folder structure

## Tech Stack

### Frontend

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3

### Libraries

- Ant Design Icons
- EmailJS

## Installation

Clone the repository

```bash
git clone https://github.com/Madhureddy9390/portfolio.git
```

Navigate into the project

```bash
cd portfolio
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview the production build

```bash
npm run preview
```
